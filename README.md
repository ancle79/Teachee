# Teachee - 课程录制软件

一个功能强大的课程录制软件，支持自定义鼠标样式，让你的教学视频更加生动有趣。

## 功能特性

- 🎬 **屏幕录制** - 支持录制整个屏幕或特定窗口
- 🎤 **音频录制** - 支持录制麦克风声音（带回声消除和降噪）
- 🖱️ **自定义鼠标样式** - 默认教鞭样式，支持多种鼠标图案选择
- 📁 **上传自定义图片** - 可以上传自己的图片作为鼠标样式
- 📏 **调节鼠标大小** - 支持 24-128px 范围调节
- ✨ **实时效果** - 录制时实时显示自定义鼠标
- 💾 **视频保存** - 录制完成后可保存为 MP4 或 WebM 格式（默认 MP4）
- 🎨 **白板功能** - 支持打开图片或创建空白页，在上面写字、绘画

## 支持的鼠标样式

### 预设样式
- 🔵 教鞭（蓝）- 默认样式
- 🔴 教鞭（红）
- 🟢 教鞭（绿）
- 🟡 教鞭（黄）
- ✨ 魔法棒
- ✏️ 铅笔
- 🖍️ 荧光笔
- ➤ 箭头

### 自定义样式
- 支持上传任意图片（PNG, JPG, GIF 等）
- 自动保存自定义光标
- 可删除不需要的自定义光标

## 安装和运行

### 安装依赖

```bash
npm install
```

### 启动应用

#### 🚀 Method 1: Quick Start (Recommended)
- Double-click `Start Teachee.bat` to launch

#### 🖥️ Method 2: Create Desktop Shortcut
1. Right-click `Start Teachee.bat`
2. Select "Send to" -> "Desktop (create shortcut)"
3. Now you can double-click the shortcut on your desktop!

#### 💻 Method 3: Command Line

```bash
npm start
```

## 使用说明

1. **选择录制源** - 在"选择录制源"面板中选择要录制的屏幕或窗口
2. **设置鼠标样式** - 在"鼠标样式"面板中：
   - 点击"📁 上传自定义图片"按钮上传你的图片
   - 拖动滑块调节鼠标大小（24-128px）
   - 选择预设样式或你的自定义图片
3. **设置音频** - 在"录制控制"面板中勾选或取消勾选"录制麦克风声音"
4. **开始录制** - 点击"开始录制"按钮开始录制
5. **停止录制** - 点击"停止录制"按钮结束录制
6. **保存视频** - 点击"保存视频"按钮将录制的内容保存到本地

### 管理自定义光标
- 上传图片后会自动保存
- 可以删除不需要的自定义光标（点击右上角的 🗑️ 按钮）
- 自定义光标会在下次启动时自动加载

## 白板功能使用说明

### 打开白板
- 点击主界面右上角的 "🎨 打开白板" 按钮
- 会打开一个独立的白板窗口

### 白板工具
- **📷 打开图片** - 打开本地图片文件作为背景
- **📄 新建空白页** - 创建新的空白画布
- **💾 保存** - 将白板内容保存为图片
- **🖊️ 画笔** - 普通画笔工具
- **🖍️ 荧光笔** - 半透明荧光笔效果
- **🧹 橡皮** - 擦除内容
- **↩️ 撤销** - 撤销上一步操作
- **↪️ 重做** - 重做已撤销的操作
- **🗑️ 清空** - 清空整个画布

### 颜色和粗细
- 18种预设颜色可选
- 支持自定义画笔粗细（1-30px）
- 6种背景颜色可选（白色、米色、浅橙色、浅蓝、浅绿、浅粉）

## 技术栈

- Electron - 桌面应用框架
- React - UI 框架
- MediaRecorder API - 录制功能
